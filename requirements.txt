# ===== INSTALLATION NOTES =====
#
# 1. GPU Support:
#    - Install CUDA-compatible PyTorch for GPU acceleration
#    - Check your CUDA version: nvidia-smi
#    - Use appropriate index URL for your CUDA version
#
# 2. PaddlePaddle Installation:
#    - For GPU: pip install paddlepaddle-gpu
#    - For CPU: pip install paddlepaddle
#
# 3. Memory Requirements:
#    - Minimum 8GB RAM for basic operation
#    - 16GB+ recommended for batch processing
#    - GPU memory: 6GB+ recommended
#
# 4. Model Downloads:
#    - Models will be automatically downloaded on first use
#    - Ensure stable internet connection for initial setup
#    - Total model size: ~2-3GB
#
# 5. Platform-Specific Notes:
#    - Windows: Ensure Microsoft Visual C++ Redistributable is installed
#    - macOS: May require Xcode Command Line Tools
#    - Linux: Ensure system packages for OpenCV (libgl1-mesa-glx, etc.)
#
# ===== QUICK INSTALLATION =====
# 
# Standard installation:
# pip install -r requirements.txt
#
# With GPU support (CUDA 11.8):
# pip install -r requirements.txt --extra-index-url https://download.pytorch.org/whl/cu118
#
# CPU-only installation:
# pip install -r requirements.txt --extra-index-url https://download.pytorch.org/whl/cpu
