PhotoValidator - Image Input Folder
========================================

This folder is where you place images for processing.

Supported formats: JPG, PNG, TIFF, BMP, WebP
The system will automatically process all images in this folder.

Results will be saved to the Results/ folder:
- Results/valid/ - Images that passed all checks
- Results/invalid/ - Images with detected issues
- Results/manualreview/ - Images requiring manual review

Instructions:
1. Copy your images to this folder
2. Run: python main_optimized.py
3. Check the Results/ folder for organized output

For more information, see the main README.md file.
